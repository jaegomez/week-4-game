Programming Collector

Rules


You will be given a random number at the start of the game.

There are four programming languages below. By clicking on a programming language you will add a specific amount of points to your total score.

You win the game by matching your total score to the random number, you lose the game if your total score goes above the random number.

The value of each programming language is hidden from you until you click on it.

Every time the game restarts, the game will change the values of each programming language.
